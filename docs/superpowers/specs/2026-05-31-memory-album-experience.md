# 回忆图鉴体验升级设计说明

## 目标

将现有“照片翻页”和“听故事模式”整合为一本面向老人本人优先设计的可听人生影集。图鉴的第一价值是陪伴与观看体验，时间线只作为次级定位能力。

## 已确认决策

- 采用“可听的人生影集”作为主体，吸收人生时间轴和家庭共编档案的优点。
- 第一优先服务老人本人：大照片、少操作、明显的一键朗读、低认知负担。
- 打开图鉴后先展示封面，保留进入一本影集的仪式感。
- 封面照片优先顺序：重点照片、家庭照片、其他非头像照片、老人头像兜底。
- 朗读不再跳转到另一套资料页；进入故事后，一次只展示一张照片、一段简短故事和当前故事播放按钮。
- 时间线作为二级目录入口，用于快速定位经历，不抢占影集第一屏。
- 图鉴中彻底移除所有家属补充提示。资料编辑只保留在设置中的数据预录入模块。
- 图鉴朗读与陪伴聊天共用 `/api/tts/synthesize`，默认音色为 `wanqing`。不再使用 Windows 系统语音或浏览器 `speechSynthesis`。
- 影集副标题固定为“慢慢翻，也慢慢听”，不再拼接照片、亲友、经历数量，也不直接使用照片备注。
- 不使用生成式图片替代真实家庭照片。后续如需生成素材，仅用于低干扰纹理和空状态插画。

## 页面结构

### 1. 影集封面

- 使用推荐封面照片铺满主要区域。
- 覆盖老人姓名、简短副标题与影集引导语。
- 主按钮为“开始翻阅”，点击后进入阅读页。

### 2. 统一阅读页

- 顶部提供返回封面、影集标题、时间线入口与刷新入口。
- 顶部只展示当前章节名称，可打开简洁章节菜单跳转。
- 每次只展示当前故事：大图、时间地点、标题、最多两句正文。
- 底部只保留上一页、朗读当前故事、下一页。
- 朗读仅由用户点击触发，不自动连续播放整本影集。

### 3. 时间线目录

- 以时间、标题、简短内容展示重要经历。
- 可展示关联照片缩略图。

## 非目标

- 首版不重写图鉴生成算法。
- 首版不增加家属账号体系或多人协同。
- 首版不新增远端图片存储。
- 首版不在图鉴中暴露资料编辑入口。
